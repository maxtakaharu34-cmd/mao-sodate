# 魔王そだて

暗黒大魔王を育てる、放置系かっこいい育成Webアプリ。

- 卵 → ベビー魔王 → こ魔王 → 成魔王 → 暗黒大魔王 の5段階進化
- 「えさ・あそぶ・きたえる・ほめる」の4操作
- 空腹/機嫌/日数を管理。放置すると弱るが、オフライン中も時間は進む（24時間上限）
- WebAudio 合成 SE、LocalStorage でセーブ

## 遊ぶ

## 進化段階

| Lv | 段階 |
|---|---|
| 1  | 呪われた卵 |
| 5  | ベビー魔王 |
| 20 | こ魔王 |
| 50 | 成魔王 |
| 100 | 暗黒大魔王 |
